Spreadly - Spreadsheet Desktop Application

Introduction

Spreadly is the spreadsheet solution for the Application Suite.

There are many features to this free software that will be explained in this file, they are

  • Cell Panel
  • Status Bar
  • Toolbar
  • Cell Name Value and Contents
  • Undo Actions
  • Save and Load Spreadsheet

Cell Panel

The cell panel is located in the main portion of Spreadly. It allows you to select a cell and then change the value through the Cell Name Value and Contents items above. You can visually see what cell you have selected by the black box.

As you put content into cells, the positions on the spreadsheet panel will reflect these changes. You can use this to your advantage, by putting titles in the cell above a cell you want a name for.

Status Bar

The status bar is located at the bottom of the Spreadly window. It shows you what you are currently doing, tips on what to do next, and what you have done. The Status bar should be your go-to item should you become lost or confused about what you have done, or how to continue.

For serious messages or errors, they will not be displayed in the status bar. These types of messages will be shown in a pop-up, giving further instruction.

Toolbar

The toolbar is located at the very top of Spreadly. This is where all of your normal toolbar-associated actions can be performed. This includes

  • File Submenu
    • Create a new Spreadsheet
    • Open an existing spreadsheet
    • Save the current Spreadsheet in it's current location
    • Save the current Spreadsheet in a new location
    • Close this Spreadsheet
  • Help Submenu
    • Open this help documentation (you found me!)
    • About the current version of Spreadly
  • Quick Action Buttons
    • Save - saves in the current location
    • Undo - undo the previous actions

Clicking any of these menus will perform the action indicated, though some menus will be greyed out when they under current circumstances, wouldn't perform any action.

Cell Name Value and Contents

The Cell Name, Value, and Contents text boxes are located right below the toolbar. These are context sensative to what you have selected in the Spreadsheet Panel and will reflect the selected cell's value, name, and content.

By clicking the contents text box, you can set the contents of the selected cell. Simply enter a formula, number, or words and hit enter. A formula is signified by an = sign at the beginning.

Undo Actions

The undo button is located in the Toolbar at the top of Spreadly. It is an orange back arrow. It is used to undo previous changes you have made to your spreadsheet since opening/creating it. One click will correspond with one change you made.

Once you have undone every action, the button will become disabled until more actions are performed.

Save and Load Spreadsheet

Save and open are located in the file submenu of the Toolbar, and there is also a quicksave button to the right.

If you click open, a window will pop up asking you to select the Spreadsheet file you wish to open. Navigate to the file, click on it, and click open. This will load the spreadsheet.

Clicking save from a spreadsheet you have saved before will simply save to the current location of the file, this will be indicated on the Status Bar. If you haven't saved before, or if you click Save As, a window will pop up allowing you to choose where to save the spreadsheet and under what name. Choose a location, name the file, and click save. The Status Bar should then indicate the save was successful.
